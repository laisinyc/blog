## HTML Document Structure

The `<html>` tag represents the **root** of an HTML document and serves as the container for all other HTML elements (except for the `<!DOCTYPE>` declaration).

**Important**: Always include the `lang` attribute to declare the language of the web page, which assists search engines and browsers.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Title of the document</title>
  </head>
  <body>
    The content of the document...
  </body>
</html>
```

## Frequently Used HTML Elements

### Radio Buttons (Single Selection)

Use the `name` attribute to group related radio buttons together. Only one option can be selected within a group.

```html
<!-- Single selection group -->
<input type="radio" name="single" value="abc">
<input type="radio" name="single" value="123" checked>
<input type="radio" name="single" value="xxx">
```

### Checkboxes (Multiple Selection)

Checkboxes allow users to select multiple options from a group.

```html
<!-- Multiple selection group -->
<input type="checkbox" name="multi" value="abc">
<input type="checkbox" name="multi" value="123">
<input type="checkbox" name="multi" value="xxx">
```

**Key Attributes:**

- `name`: Groups related inputs together
- `value`: Defines the actual value submitted with the form
- `checked`: Sets the default selected state

### Dropdown Menus

The `<select>` element creates a dropdown menu with `<option>` elements.

```html
<select name="province">
  <option>Beijing</option>
  <option>Nanjing</option>
  <!-- When value attribute is set, it overrides the displayed text -->
  <option value="sh">Shanghai</option>
</select>
```

**Important Notes:**

- Text between `<option>` tags is the display value
- If no `value` attribute is set, the displayed text becomes the submitted value
- The `value` attribute takes precedence over the displayed text when the form is submitted
